# Strava 一键点赞插件使用指南

## 插件介绍

**GCW and Strava Kudo All** 是一款专为 Strava 用户设计的 Chrome 浏览器插件，可以帮助你快速给关注者的动态点赞，大大提升互动效率。

### 插件特点

- ✅ **一键点赞**：点击即可给当前页面所有动态点赞
- ✅ **智能识别**：自动识别并跳过已经点赞的动态
- ✅ **完全免费**：开源免费，无广告
- ✅ **操作简单**：界面简洁，使用方便

## 安装步骤

### 方法一：从 Chrome 网上应用店安装（推荐）

1. 打开 Chrome 浏览器，访问 [插件页面](https://chromewebstore.google.com/detail/gcw-and-strava-kudo-all/folhiecfhnmdniibjjcfogpdoafdamoc)
2. 点击页面右上角的 **"添加至 Chrome"** 按钮
3. 在弹出的确认对话框中点击 **"添加扩展程序"**
4. 安装完成后，浏览器右上角会出现插件图标

### 方法二：手动安装（开发者模式）

如果你无法访问 Chrome 网上应用店，可以手动安装：

1. 下载插件的 ZIP 压缩包
2. 解压到本地文件夹
3. 打开 Chrome 浏览器，进入 `chrome://extensions/`
4. 开启右上角的 **"开发者模式"**
5. 点击 **"加载已解压的扩展程序"**
6. 选择解压后的插件文件夹

## 使用方法

### 基本使用

1. 打开 [Strava 官网](https://www.strava.com/) 并登录你的账号
2. 进入 **"动态"** 页面（Dashboard）或任意好友的动态页面
3. 点击浏览器右上角的插件图标
4. 在弹出的面板中点击 **"Kudo All"** 按钮

### 操作说明

插件会自动扫描当前页面的所有动态，给未点赞的动态自动点赞。点赞完成后会显示点赞数量提示。

## 插件设置

**GCW and Strava Kudo All** 插件设计简洁，无需复杂设置：

- **智能跳过**：默认自动跳过已点赞的动态
- **点赞反馈**：活动点赞和手动一致无特殊提示

## 注意事项

### 使用规范

1. **适度使用**：不要过于频繁地批量点赞，以免触发 Strava 的反垃圾机制
2. **遵守规则**：尊重 Strava 社区规范，不要滥用点赞功能
3. **隐私保护**：插件不会收集或存储你的个人信息

### 常见问题

**Q: 插件不工作怎么办？**

A: 请检查以下几点：
- 确保已登录 Strava 账号
- 确保在动态页面使用插件
- 尝试刷新页面后重新使用
- 检查插件是否需要更新

**Q: 点赞后动态会取消吗？**

A: 不会，插件只会给未点赞的动态点赞，不会取消已有的点赞。

**Q: 是否支持其他浏览器？**

A: 目前仅支持 Chrome 浏览器，其他 Chromium 内核浏览器（如 Edge）可能兼容。
