## 二、功能模块

### 1. 规格查询

支持公制螺纹(M)和英制管螺纹(G)的详细规格查询：

- **公制螺纹**：M3 ~ M64
- **英制管螺纹**：G1/8 ~ G3

输出信息包括：
- 大径、螺距、攻丝底孔直径
- 螺纹牙型参数（H、h/8、h/4、实际牙高）
- 3D 打印优化尺寸（外牙大径、内牙大径）

### 2. 公制螺丝表格

显示公制螺纹基本尺寸表（GB/T 196）：

| 列名 | 说明 |
|------|------|
| 螺纹代号 | M3 ~ M30 |
| 大径 | 螺纹大径 |
| 螺距 P | 螺距 |
| H=0.866P | 等边三角形高度 |
| 尖端裁切 h/8 | 螺纹尖端裁切量 |
| 底部嵌入 h/4 | 底部嵌入螺柱量 |
| 实际牙高 | 实际牙型高度 |
| 3D外牙大径 | 3D打印外螺纹优化尺寸 |
| 3D内牙大径 | 3D打印内螺纹优化尺寸 |

### 3. 英制螺丝表格

显示英制螺纹和英制管螺纹基本尺寸表：

- **英制螺纹**：#2-56 ~ 1/2"-13
- **英制管螺纹**：G1/8 ~ G1½（GB/T 7307）

### 4. 规格对照表

常用英制规格与公制规格对照，标注兼容性：

| 英制规格 | 最接近公制 | 兼容性 |
|----------|------------|--------|
| 1/4"-20 | M6×1.0 | 紧急替代 |
| #10-24 | M5×0.8 | 不可互换 |

### 5. 圈数计算

计算 3D 打印螺纹圈数与角度：

**输入**：
- 螺纹类型（公制/英制）
- 螺纹规格
- 螺丝高度(长度)

**输出**：
- 所需圈数
- 总旋转角度
- 完整圈数
- 余数角度

## 三、数据结构

### 1. 公制螺纹数据 (`metricThreadData`)

```javascript
{
  'M6': {
    major: 6.000,        // 大径
    pitch: 1.0,          // 螺距
    tapDrill: 5.0,       // 攻丝底孔直径
    nutThickness: 4.8,   // 螺母厚度参考
    male3DPrint: 5.95,   // 3D打印外牙大径
    female3DPrint: 6.05  // 3D打印内牙大径
  }
}
```

**支持规格**：M3, M4, M5, M6, M8, M10, M12, M14, M16, M18, M20, M22, M24, M27, M30, M36, M42, M48, M56, M64

---

### 2. 英制管螺纹数据 (`imperialThreadData`)

```javascript
{
  'G1/2': {
    od: 20.955,           // 管外径
    tpi: 14,              // 每英寸牙数
    pitch: 1.814,         // 螺距
    major: 20.955,        // 大径
    pitchDiameter: 19.793,// 中径
    minor: 18.631         // 小径
  }
}
```

**支持规格**：G1/8, G1/4, G3/8, G1/2, G3/4, G1, G1¼, G1½, G2, G2½, G3

## 四、计算公式

### 1. 螺纹牙型参数

```
H = 0.866 × P           // 等边三角形高度
尖端裁切 = H / 8         // 螺纹尖端裁切量
底部嵌入 = H / 4         // 底部嵌入螺柱量
实际牙高 = H - H/8 - H/4 // 实际牙型高度
小径 = 大径 - 2 × 实际牙高
```

### 2. 圈数与角度计算

```
圈数 = 高度 ÷ 螺距
总角度 = 圈数 × 360°
完整圈数 = floor(圈数)
余数角度 = (圈数 - 完整圈数) × 360°
```
