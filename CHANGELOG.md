## [1.2.0] - 2026-07-23

### ✨ 新增

#### 📊 卡片交互
- **距离卡片**点击弹窗设置距离标记间隔（0=关闭，1-50km）
- **爬升卡片**点击弹窗设置隧道/桥梁过滤参数
- **时长卡片**点击弹窗设置平均速度（5-60 km/h）
- **兴趣点卡片**点击弹窗查看 POI 列表，一键定位到地图

#### 📍 兴趣点增强
- 上传 GPX 时自动加载 `<wpt>` 航点为地图标记
- **兴趣点去重**：导入时按坐标+名称去重，避免重复
- 新增兴趣点统计卡片（第四张统计卡片）

#### 🗺️ 路书链接自动识别
- 上传 GPX 时自动从 `<metadata><link>` 提取 Strava / iGPSPORT / Garmin 链接
- 自动填入对应输入框并激活按钮

### 🐛 修复
- **过滤开关视觉状态**：开关 toggle 的 CSS specificity 冲突导致显示与实际状态不一致
- **过滤提示数值**：修正 `原始值 - 过滤值` 计算顺序（之前写反导致显示负数）
- **距离标记默认值**：从 1km 改为 0（默认关闭）
- **重复航点**：导入含重复 `<wpt>` 的 GPX 时不再重复添加

### 🗑️ 移除
- 活动描述编辑区
- 分享面板（微博/Twitter/复制链接）
- 路书链接手动输入改为自动识别（保留手动输入框）

## [1.1.0] - 2026-07-23

### ✨ 新增

#### 🎨 轨迹颜色编码
- 按海拔渐变着色（蓝→青→绿→黄→红）
- 按坡度渐变着色（蓝色下坡→红色上坡）
- 地图底部颜色模式切换栏
- 右侧颜色图例（显示数值范围）

#### ⛰️ 海拔剖面交互
- 鼠标悬浮显示十字线 + 距离/海拔提示气泡
- 悬浮时地图同步显示对应位置标记
- 点击海拔图某点，地图自动定位并缩放

#### ✏️ 轨迹编辑
- 编辑工具栏（地图左上角）
- ↩ 撤销 / ↪ 重做（支持所有编辑操作）
- ➕ 添加节点：点击轨迹任意位置插入
- 🗑️ 删除节点：显示所有节点手柄，点击删除
- 拖拽节点重新规划路线

#### 📏 距离测量
- 工具栏 📏 按钮激活测距模式
- 点击放置测距点，虚线连接
- 实时显示总距离
- 双击结束测量

#### 📌 右键菜单
- 📍 放置标记
- 📏 从这里测距
- 📋 复制坐标
- 🎯 居中地图

#### 📍 距离标记
- 筛选面板中开启，沿轨迹每隔 1km 显示编号标记

#### 🖥️ 坐标显示
- 地图左下角实时显示鼠标经纬度坐标

#### 📷 POI 增强
- **自动吸附**：添加 POI 时自动匹配最近的轨迹点
- **GPX 导出含兴趣点**：导出 GPX 文件时自动写入 `<wpt>` 节点
- 弹窗显示吸附距离

#### ⚙️ 速度设置
- 新增平均速度调节（默认 25 km/h，可调 5-60）
- 实时更新预估时长

#### 🗺️ 地图增强
- 默认中心点改为成都（30.5728, 104.0668）
- 地图视图加宽（70vh）
- Leaflet 本地离线化，支持无网络使用

### 🗑️ 移除
- 电子骑友（虚拟骑行模拟）功能
- 速度着色模式（保留海拔/坡度）

## [1.0.0] - 2026-07-22

### 🚀 首次发布

#### GPX 轨迹渲染
- 支持 GPX 文件上传（点击 + 拖拽）
- 地图自动缩放适应轨迹范围
- 起点（S）/ 终点（E）标记
- 街道 / 卫星底图切换（地图右上角）
- 右下角实时显示鼠标经纬度

#### 📷 机位踩点 (POI)
- 在地图上标记拍摄点 / 兴趣点
- 6 种 POI 类型：📸 拍照点 / 🚻 厕所 / 🪑 休息 / 🏪 补给点 / 💧 饮水 / ⚠️ 注意
- 点击 POI 可删除
- POI 数据 JSON 导出 / 导入
- 工具栏位于地图左上角（工具栏 / 面板内）
- POI 自动持久化到 localStorage

#### ⛰️ 海拔剖面
- Canvas 绘制海拔图
- 距离刻度（km）
- 海拔刻度（m）
- 三列布局（距离 | 海拔 | 爬升）

#### 🚇 隧道过滤
- 基于梯度检测的 GPS 异常点过滤
- 可调坡度阈值（5%-30%）和平滑窗口（0-100）
- 可开关过滤功能
- 实时显示过滤了多少无效爬升

#### 🚴 电子骑友
- 沿轨迹匀速前进动画
- 默认 25 km/h，可调速度（5-50 km/h）
- 进度条 + 百分比
- 预估到达时间
- 起点 / 终点 / 当前位置标记（S / E / 箭头）
- 暂停 / 重置控制

#### 📊 数据统计
- 距离 (km)
- 爬升 (m)
- 时长（按 25km/h 匀速计算）

#### 🔗 路书链接
- Strava
- iGPSPORT
- Garmin Connect

#### 📤 分享
- 微博分享
- Twitter 分享
- 复制链接
- 活动标题自动设置为 GPX 文件名

#### 💾 数据持久化
- 用户名保存到 localStorage
- POI 数据保存到 localStorage
- 页面加载自动恢复上次状态
